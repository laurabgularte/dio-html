História dos computadores

(Voltar)

A história dos computadores começou por volta de 1944, com o Harvard Mark I, depois disso, com as evoluções tecnológicas, foram surgindo novas gerações de computadores. Eles foram se tornando cada vez menores e com mais capacidade de armazenamento e processamento.

O que são Clients?

(Voltar)

São os usuários(máquinas) que recebem as informações vindas dos servidores.

O que são Servers?

(Voltar)

Entregam as informações aos clientes.

Como é uma estrutura básica de um HTML?

(Voltar)

O HTML(HyperText Markup Language) é estruturado basicamente pelas seguintes tags:

  1. DOCTYPE: define o documento, sempre está na primeira linha do código
  2. Head: contém as informações que os usuários não conseguem ver, dados de uso e controle do documento
  3. Body: é a estrutura da página visível para o usuário, contém todos os textos e mídias que se deseja apresentar

Entendendo a diferença entre listas ordenadas e não ordenadas

(Voltar)

As listas ordenadas são criadas usando a tag OL e é utilizada quando queremos definir uma ordem para os itens listados. Já as listas não ordenadas, que usam a tag UL, são utilizadas quando os itens podem ser listados em qualquer ordem, pois sua posição não altera a significação da lista.
